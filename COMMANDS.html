BOT COMMANDS

🔨 Moderation

%ban Explained
Ban User From Server
Example:
%ban Mention User
%kick Explained
Kick User From Server
Example:
%kick Mention User
%mute Explained
Mute User For A Set Duration
Example:
%mute Mention User 1s/m/h/d
%unmute Explained
Unmute User
Example:
%unmute Mention User
%lockdown Explaind
Lock A text Channel, So No One Can Type
Example:
%lockdown #ChannelName
%lockdown unlock Explained
Unlock The Locked Text Channel
Example:
%lockdown unlock #ChannelName
%purge Explained
Deletes A Specified Amount Of Message In A Channel
Example:
%purge 50
%purge all
